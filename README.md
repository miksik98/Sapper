# Sapper
A well-known game written in Scala. GUI in Swing.

How to run it:
-----------
1. Compile:
    
    mkdir bin
    scalac -d bin src/IntroPanel.scala src/Minesweeper.scala src/MinesweeperWindow.scala

2. Run:
    
    scala -classpath bin GameMain
